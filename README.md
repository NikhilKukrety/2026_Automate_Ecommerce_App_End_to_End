Developed a comprehensive end-to-end automation framework for an e-commerce application using industry-standard frameworks and design patterns. Integrated GitHub Copilot to accelerate development, improve code consistency, and support seamless project integration.

Tech Stack:
- Java
- Selenium WebDriver
- Maven
- TestNG
- Page Object Model (POM)
- Cucumber Integration (BDD)

Features:
- BaseTest and Hooks for driver lifecycle
- BasePage for reusable methods
- Explicit wait handling
- Data Driven Testing using Scenario Outline
- TestNG Listeners Integration for Screenshot handling in case of failures
- Modular design
- Dynamic values handling using config file
- Cucumber and TestNG Reporting Mechanism
- CI Integration with Jenkins
- Email Reporting to the team

Future Scope:
- Cucumber Reports in Jenkins
- Browser Headless Mode
- TestNG.xml for Grouping
